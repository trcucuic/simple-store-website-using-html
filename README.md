# Elegant Accessories Website

## Project Description

This project is a simple **accessories store website** created using HTML.
The website introduces a fictional store called **Elegant Accessories** that sells stylish fashion accessories.

## Features

* Simple homepage introducing the accessories store.
* List of popular accessories such as necklaces, bracelets, rings, and earrings.
* Navigation links to explore more products and another page.
* Email link to contact the store.
* Featured accessory image.
* Customer service contact table.

## Technologies Used

* HTML5

## Website Sections

The webpage includes several sections:

1. **Store Introduction**
   A short description welcoming visitors to the accessories store.

2. **Popular Accessories List**
   A list showing the main products sold by the store.

3. **Navigation Links**

   * Explore accessories online.
   * Open the second page of the website.
   * Send an email to the store.

4. **Featured Product Image**
   Displays a sample accessory image.

5. **Customer Service Table**
   Shows contact emails for customer support and orders.

## Project Files

* `FirstPage.html` – Main page of the accessories website
* `SecondPage.html` – Product page
* `README.md` – Project documentation
